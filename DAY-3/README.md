# SERVER IN VISUAL STUDIO CODE EDITOR

> Unlike various other robust IDE like eclipse, intelliJ, NetBeans etc. Visual Studio Code doesn't support running server own its own but tomcat apache, websphere can now be added with the help of Community Sever Connectors which is an extension or plugin in vscode which connect vscode with open source community driven server and runtimes.Try adding it to your visual studio code.
